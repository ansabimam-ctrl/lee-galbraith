# Lee Galbraith - Professional Portfolio V1.0

A production-grade Next.js application showcasing assets across character scopes.

## Feature Stack
- **Model Directories**: Modular data routing mapped across Look configurations 1–7.
- **Interaction Sets**: Lightbox overlays scaling assets appropriately.
- **Access Support**: Streamlined JotForm connections.

## Quickstart Procedures
```bash
npm install
npm run dev
```
